## 🚀 Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm (v9 or higher)
- Angular CLI (v19)

### Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd ecommerce-angular
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
ng serve
```

4. View the application:

```
http://localhost:4200
```
